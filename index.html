Introduction

Sweden has large areas of both intact and drained wetlands. This project is a manual for creating a database of Swedish wetlands and then analyse and model these wetlands using Karttur's Geoimagine Framework. The aim of the project is to improve the understanding of the Greenhouse Gas (GHG) balance of Swedish wetlands. This section ("blog") contains the more verbose manuals, the json command files to run the project are under the parallell section geoimagine03-proj-wetland-se-json.
